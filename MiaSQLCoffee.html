Here I create a mock data set with random values. I am currently interested in coffee and anxiety so my dataset is based around this theme. The values are not based on reality and therefor the results are unreliable as a source, but the design would make for an interesting data collection project. Later, I will show you how to tweak the data set and run a full data analysis using Python.

My mock data set is structured as follows:

  1. client_id: Unique identifier for each individual.
  2. gender: Male or Female.
  3. age_group: Age groups like 18-25, 26-35, etc.
  4. caffeine_dosage_mg: The amount of caffeine intake in milligrams.
  5. hours_after_intake: Hours passed after caffeine intake.
  6. anxiety_level: A score (let's say between 0 to 10) indicating the anxiety level.

I will show you step by step how to create a mock data set so that you can practice on your own mock data!

import random
import sqlite3
import pandas as pd

# Function to generate mock data
def generate_mock_data(num_clients=100):
    data = []
    genders = ['Male', 'Female']
    age_groups = ['18-25', '26-35', '36-45', '46-55', '56+']
    
    for client_id in range(1, num_clients + 1):
        gender = random.choice(genders)
        age_group = random.choice(age_groups)
        caffeine_dosage_mg = random.randint(50, 300)  # Random caffeine dosage between 50mg to 300mg
        
        previous_anxiety_level = random.randint(5, 10)  # Initial anxiety level
        
        # Generating data for each hour after intake up to 5 hours
        for hour in range(1, 6):
            # Simulate the decrease in anxiety level over time with some randomness
            change = random.randint(-3, 2)
            anxiety_level = max(0, min(10, previous_anxiety_level + change))
            
            data.append((client_id, gender, age_group, caffeine_dosage_mg, hour, anxiety_level))
            
            previous_anxiety_level = anxiety_level
    
    return data

Adjusting the data based on a study¶

The study "Caffeine Intake and Its Sex-Specific Association with General Anxiety: A Cross-Sectional Analysis among General Population Adults" indicates that women have higher anxiety levels after caffeine consumption. We will adjust our data set based on this study, while keeping in mind that most of the values are random and not based on real-life. With enough data, we could potentially fine-tune the data set to be highly realistic but that is not our goal.

Here's the adjusted approach:

  1. Base Anxiety Level: We'll set a general base anxiety level.
  2. Adjustment for Caffeine Intake: For women, we'll increase the anxiety level based on the caffeine intake, especially for those in the higher tertiles of intake.
  3. Age-Related Adjustments: Although the exact relationship between age and caffeine-related anxiety is not provided in the study, we'll introduce a subtle age-related adjustment. Older individuals (e.g., age group '56+') might have slightly higher anxiety levels than younger age groups.
  4. Random Variation: As before, we'll introduce some random variation to make the dataset more realistic.

import numpy as np


def generate_study_based_mock_data(num_clients=1000):
    data = []
    genders = ['Male', 'Female']
    age_groups = ['18-25', '26-35', '36-45', '46-55', '56+']
    
    # Proportions based on study (74.1% women)
    gender_distribution = ['Female'] * 741 + ['Male'] * 259
    
    for client_id in range(1, num_clients + 1):
        gender = random.choice(gender_distribution)
        age_group = random.choice(age_groups)
        
        # Adjusted caffeine intake based on study
        if gender == 'Female':
            caffeine_dosage_mg = int(np.random.normal(212.4, 159.6))
        else:
            caffeine_dosage_mg = int(np.random.normal(243.8, 177.7))
        
        caffeine_dosage_mg = max(50, min(caffeine_dosage_mg, 400))  # Ensure it's within a reasonable range
        
        hours_slept = random.randint(3, 10)  # Random hours slept between 3 to 10 hours
        
        # Base anxiety level
        base_anxiety = 5.0
        
        # Adjust for caffeine intake
        if gender == 'Female' and caffeine_dosage_mg > 200:  # Assuming this as the highest tertile for simplicity
            base_anxiety += 1.5  # Slightly increased effect for study findings
        
        # Age-related adjustment (subtle changes for simplicity)
        if age_group == '56+':
            base_anxiety += 0.5
        
        previous_anxiety_level = random.randint(int(base_anxiety - 2), int(base_anxiety + 2))
        
        # Generating data for each hour after intake up to 5 hours
        for hour in range(1, 6):
            # Simulate the decrease in anxiety level over time with some randomness
            change = random.randint(-3, 2)
            anxiety_level = max(0, min(10, previous_anxiety_level + change))
            
            data.append((client_id, gender, age_group, caffeine_dosage_mg, hour, anxiety_level, hours_slept))
            
            previous_anxiety_level = anxiety_level
    
    return data
